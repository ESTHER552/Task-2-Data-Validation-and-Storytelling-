# Retail Sales & Profit Overview Dashboard
This Power BI project visualizes sales and profit performance across regions, categories, and segments using the Sample Superstore dataset.

## Key Insights
- Highest sales recorded in the **South region** for **Technology** category.
- **Standard Class** shipping mode generated most orders.
- **Home Office** segment contributed over 50% of total sales.

## Tools Used
- Microsoft Power BI

## Features
- Interactive charts and slicers
- Drill-down by category, region, and segment
- Dynamic KPIs: Average and Maximum Sales
